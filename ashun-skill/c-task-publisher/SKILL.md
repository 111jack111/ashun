# 【C】任务 - 公众号爆款日更

自动化公众号发布流程，把发布变成流水线。

---

## 核心能力

- 6 阶段自动化：解析→创作→审查→排版→发布→通知
- 朱砂红排版模板（#C41E3A）
- 自动敏感词审查
- 微信 API 自动发布
- 飞书通知推送

---

## 使用方式

```
执行【C】任务

用户分享截图/文字/链接 → 自动解析 → 创作 → 审查 → 排版 → 发布到公众号草稿箱 → 飞书通知
```

---

## 执行流程

### 阶段 1：解析素材
输入：用户分享截图/文字/链接
输出：核心观点 + 金句 + 案例 + 数据

### 阶段 2：创作
输出：2000-5000 字顺哥风格文章

### 阶段 3：自我审查
输出：审查报告（通过才能继续）

### 阶段 4：排版
输出：HTML 排版文件（朱砂红模板）

### 阶段 5：发布
输出：公众号草稿箱 + 链接

### 阶段 6：通知
输出：飞书推送（标题 + 链接 + 审查结果）

---

## 排版参数（朱砂红模板 v1.3）

- 大标题：朱砂红 #C41E3A，32px，Times New Roman
- 章节标题：朱砂红 #C41E3A，18px，无框
- 分隔线：淡朱砂红 rgba(196,30,58,0.2)
- 正文：14px，无句号结尾
- 背景：纯白 #FFFFFF
